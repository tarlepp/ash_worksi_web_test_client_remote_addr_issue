# What is this?

Someone didn't get `REMOTE_ADDR` working with Symfony `WebTestCase`, so
I created this simple repository to demonstrate that is can be done.

## How to test this?

1. Clone repository
1. Run `composer install`
1. Run `php bin/phpunit`
1. Profit
